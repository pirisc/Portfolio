**Comida / NoComida**

Comida/NoComida is a text classification project about ~1000 text phrased labels with `comida` and `no_comida`. The goal of the project is to deploy an app with Gradio where you enter a phrase and is label.

The dataset for this project was created by my and you can find it [here](https://huggingface.co/datasets/Sairii/comida_no_comida)

Also the final gradio app is deployed on Hugging Face and you can find it [here](https://huggingface.co/spaces/Sairii/learn_hf_food_not_food_text_classifier_demo)
